Welcome!

This is the home page of a Django project with dynamic routing based on file structure.

How it works?

  • File templates/pages/index.html → URL: /
  • File templates/pages/about.html → URL: /about/
  • File templates/pages/contact.html → URL: /contact/
  • File templates/pages/blog/index.html → URL: /blog/
  • File templates/pages/blog/first-post.html → URL: /blog/first-post/

Just create HTML files in the templates/pages/ folder and they will automatically become available at corresponding URLs!
